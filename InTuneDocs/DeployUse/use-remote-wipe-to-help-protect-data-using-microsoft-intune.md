# 協助透過使用 Microsoft Intune 的完整或選擇性抹除來保護您的資料
如同裝置一般，有些時候，您會想要或需要將已部署至電腦與行動裝置的[應用程式淘汰](retire-apps-using-microsoft-intune.md)，因為您不再需要它們。 您也可能會想要從裝置移除公司資料。 若要這樣做，Intune 會提供選擇性抹除和完整抹除功能。 由於行動裝置可以儲存公司機密資料，也可以存取許多公司資源，因此您可以從 Intune 發出遠端裝置抹除命令來抹除遺失或遭竊的裝置。 此外，使用者可以在已於 Intune 中註冊的私人擁有裝置上，從 Intune 發出遠端裝置抹除命令。

  > [!NOTE]
  > 本主題只會說明如何抹除受 Intune 管理的裝置。 您也可以使用 [Azure 預覽入口網站](https://portal.azure.com) 來 [從應用程式抹除公司資料](wipe-managed-company-app-data-with-microsoft-intune.md)

## 完整抹除


[完整抹除] 會將裝置還原為其原廠預設值，移除所有的公司和使用者資料與設定， 並從 Intune 移除裝置。 在提供裝置給其裝置遺失或遭竊的新使用者或執行個體之前，完整抹除適用於重設裝置。  請小心選取完整抹除。

## 在裝置上的資料無法復原

選擇性抹除 選擇性抹除會移除公司資料，包括適用的行動應用程式管理 (MAM) 資料、設定和裝置的電子郵件設定檔。 選擇性抹除會將使用者的個人資料保留在裝置上。 並從 Intune 移除裝置。

### Windows 10 不支援對已加入 Azure Active Directory 的裝置進行選擇性抹除

1.  從 Intune 管理主控台遠端抹除裝置 選取要抹除的裝置。

    -   **您可以依使用者或依裝置來尋找裝置。**

        1.  依使用者：

        2.  在 [Intune 管理主控台](https://manage.microsoft.com/)中，選擇 [群組]  &gt;  [所有使用者] 選擇您要抹除其行動裝置的使用者名稱。

        3.  選擇 [檢視內容] 在使用者的 [內容] 頁面上，選擇 [裝置]，然後選擇您要抹除之行動裝置的名稱。

    -   **使用 Ctrl 並按一下滑鼠左鍵來複選裝置。**

        1.  依裝置：

      ![在 [Intune 管理主控台](https://manage.microsoft.com/)中，選擇 [群組]  &gt;  [所有行動裝置]](../media/dev-sa-wipe.png)

        2.  啟動淘汰或抹除作業 選擇 [裝置]，然後選擇您要抹除之行動裝置的名稱。

2.  使用 Ctrl 並按一下滑鼠左鍵來複選裝置。

3.  選擇 [淘汰/抹除]

    -   此時會顯示一則訊息，提示您確認是否要淘汰裝置。

    -   若要執行 [選擇性抹除] 僅移除公司應用程式和資料，請選擇 [是]。 若要執行 [完整抹除] 以清除所有應用程式和資料，並將裝置還原成原廠預設值，請選取 [淘汰之前抹除裝置]。 這個動作適用於 Windows 8.1 以外的所有平台

您無法復原完整抹除所移除的資料
