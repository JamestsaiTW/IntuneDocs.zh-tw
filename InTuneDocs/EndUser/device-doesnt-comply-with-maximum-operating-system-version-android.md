# Android 裝置不符合最大的作業系統版本

您的 Android 裝置不符合最大作業系統版本的 IT 系統管理員原則。 如需協助，請連絡 IT 系統管理員。 如需其連絡資訊，請查看[公司入口網站](http://portal.manage.microsoft.com)。
