# Windows 裝置不符合最大的作業系統版本

IT 系統管理員已設定您的 Windows 裝置可具有的最高作業系統版本，但您的裝置不符合該設定。 若要瞭解裝置允許的最高作業系統，您需要連絡 IT 系統管理員。 如需其連絡資訊，請查看[公司入口網站](http://portal.manage.microsoft.com)。
