# 裝置不符合最大的作業系統版本

IT 系統管理員已設定您的 Windows 裝置可具有的最高作業系統版本，但您的裝置不符合該設定。 若要瞭解裝置允許的最高作業系統，您需要連絡 IT 系統管理員。

### 請參閱
[使用具有 Intune 的 Windows 裝置](using-your-windows-device-with-intune.md)
