# iOS 裝置不符合最新的作業系統版本設定

您的 iOS 裝置不符合您可使用之最新作業系統版本的 IT 系統管理員原則。 請連絡 IT 系統管理員。 如需其連絡資訊，請查看[公司入口網站](http://portal.manage.microsoft.com)。
