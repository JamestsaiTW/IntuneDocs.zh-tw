# iOS 裝置沒有最基本必備的作業系統版本

您的 iOS 裝置不符合 IT 部門所要求的最基本作業系統版本。  更新行動裝置之前，請點選 [設定] &gt; [關於] 來檢查目前作業系統版本。 若要將裝置更新到最新版本，請點選 [設定] &gt; [一般] &gt; [軟體更新]。

是否仍需要協助？ 請連絡 IT 系統管理員。 如需其連絡資訊，請查看[公司入口網站](http://portal.manage.microsoft.com)。
