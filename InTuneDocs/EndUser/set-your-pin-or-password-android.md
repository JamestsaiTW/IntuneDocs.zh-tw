# 設定您的 PIN 或密碼

當您使用 Intune 存取公司或學校的資料時，IT 系統管理員可能會要求您在 Android 裝置上設定 PIN 或密碼。 如果看到訊息提示您設定 PIN 或密碼，請使用下列步驟。

1.  點選 [設定] &gt; **[安全性]** &gt; [螢幕鎖定] [密碼]&gt; 。

2.  選擇並確認新密碼。


是否仍需要協助？ 請連絡 IT 系統管理員。 如需其連絡資訊，請查看[公司入口網站](http://portal.manage.microsoft.com)。
