# 關閉 Microsoft 使用狀況資料收集

為了改善其產品和服務，Microsoft 會自動收集有關公司入口網站應用程式可靠性和效能以及其使用方式的匿名資料。 

您可以使用公司入口網站應用程式中的 [使用方式資料] 設定，來關閉 iOS 裝置上該資料的收集。 IT 系統管理員無權控制該資料的收集，且無法變更您對該設定的選項。
