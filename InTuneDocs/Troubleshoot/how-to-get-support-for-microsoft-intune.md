# 如何取得 Microsoft Intune 支援
 Microsoft 支援服務都會提供 Microsoft Intune 的全球性技術、售前、帳單和訂閱支援。 付費及試用訂閱可透過網路和電話兩種途徑取得支援服務。 線上技術支援提供英文與日文。 非技術支援提供其他語言。

本主題說明如何開啟許多語言的服務要求或電話支援。

如需使用 Intune 的自我協助，請參閱 [Microsoft Intune 的一般疑難排解提示](general-troubleshooting-tips-for-microsoft-intune.md) 或列在本主題之**請參閱**小節中特定問題的任何疑難排解主題。

>[!NOTE]
>
>主要客戶可以在[頂級支援頁面](https://support.microsoft.com/en-us/premier/contacts)開啟 Intune 支援票證。

## 建立線上服務要求

1.  使用 Intune 認證登入 [Office 365 系統管理中心](https://portal.office.com/admin/default.aspx#Overview_SupportPage_AltUSupportPage)。

2.  在 [建立服務要求] 下，選取您要開啟票證的問題類別：

    -   針對計費、授權以及帳戶問題，請選取 [計費和產品資訊]。

    -   如需所有其他 Intune 問題，請選取 [行動裝置管理]。

    > [!NOTE]
    > 您可能必須選擇清單底部的 [更多資訊]，以查看所有類別。

3.  在 [找出問題]  頁面上，選取您需要協助的 [功能] 和 [徵兆]。 系統會要求您提供 [問題摘要] 和 [問題詳細資料]。

4.  在 [檢閱建議]  頁面上，您會發現可協助您自行解決問題的內容連結。  如果您不能解決此問題，您可以選擇頁面底部的 [是，繼續] 提交服務要求。

5.  在 [新增詳細資料] 頁面上，提供要求的資訊。 您可以選擇 [附加檔案]，附加螢幕擷取畫面或其他文件。

6.  在 [確認並提交] 頁面上提供連絡資訊、檢閱您的要求，然後選擇 [提交要求]。 提交要求之後，會將您帶往服務要求清單。

## 電話支援
若需依國家和地區的支援電話號碼清單，或是需要知道支援時數和各地區支援的語言清單，請參閱 [Microsoft Intune 的連絡電話協助支援](contact-assisted-phone-support-for-microsoft-intune.md)。

## 追蹤服務要求
若要追蹤您的要求，請移至 [Office 365 系統管理中心](https://portal.office.com/admin/default.aspx)。 在 [建立服務要求] 清單底部，選擇 [查看所有支援票證]。 根據預設，您會看到所有服務要求處於開啟狀態。 若要變更所要查看的服務要求 (例如若要查看已關閉的要求)，請選擇頂端導覽列的篩選圖示，並選取您慣用的檢視。

> [!NOTE]
> **如果您使用 Microsoft System Center Configuration Manager** 或 **Microsoft System Center Endpoint Protection**：如需 Configuration Manager 或 Endpoint Protection 的技術支援，請連絡您的合作夥伴，或瀏覽 [Microsoft System Center Configuration Manager 和 System Center Endpoint Protection 解決方案中心](http://www.microsoft.com/en-us/server-cloud/products/system-center-2012-r2/resources.aspx)，以尋找豐富的自助資料，並可在線上或電話開啟支援要求。
>
> System Center Configuration Manager 或 System Center Endpoint Protection 的技術支援需要付費，或從您現有的授權或頂級支援協議中扣減。

## 其他支援選項

### 合作夥伴
您可以選取 Microsoft 合作夥伴並委派系統管理功能，包括建立服務要求票證。

### 大量授權
如果您已透過大量授權方案向 Microsoft 購買授權，請利用下列途徑尋求支援：

-   如需有關授權與尋找金鑰的支援，請參閱 [大量授權服務中心](http://go.microsoft.com/fwlink/p/?LinkID=282016)。

-   如有帳單問題，請參閱 [帳單和訂閱管理支援](http://support.microsoft.com/oas/default.aspx?prid=15371)。

-   如需大量授權的一般資訊，請參閱 [大量授權](http://go.microsoft.com/fwlink/p/?LinkID=282015)。

## 回應支援案例
我們會將收到的支援案例依優先順序排列，以便能夠以最快的速度來處理最緊急的案例。 為了協助排列支援案例優先順序，我們會為每個案例指定下表所示的嚴重性等級。 您也可以查看每個嚴重性等級的回應時間。

**初始回應時間：**

|嚴重性等級|作業與支援描述|Microsoft Intune/Enterprise Mobility Suite<br /><br />Premier|Microsoft Intune/Enterprise Mobility Suite<br /><br />Professional|
|------------------|--------------------------------------|------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------|
|嚴重性 A (重大)|一項或多項服務無法存取或無法使用。 生產、作業或部署期限受到嚴重影響，或是將嚴重影響生產或獲利能力。 多個使用者或服務受到影響。|1 小時|2 小時|
|嚴重性 B (高)|服務可以使用但功能減弱。 此情形的業務衝擊屬於中等，且可在上班時間處理。 單一使用者、客戶或服務受到部分影響。|2 小時|4 小時|
|嚴重性 C (中)|此情形的業務衝擊非常輕微。 發生的問題雖然重要，但對客戶目前的服務或生產力的衝擊並不顯著。 單一使用者正經歷部分中斷狀況，但有可接受的因應措施。|4 小時|8 小時|

## 支援合作關係
成功的技術支援取決於 Microsoft 支援服務與您組織的 IT 部門之間的合作關係。  在您組織內進行初步疑難排解，結合您的系統管理員與 Microsoft 之間的有效通訊，將可協助快速解決問題。

### 系統管理員可如何提供協助
具備 Intune 系統管理員角色的人員 (包括租用戶系統管理員和服務系統管理員) 有權存取 Intune 管理主控台，並可直接與 Microsoft 溝通有關 Intune 服務要求的人員。 這些系統管理員也可以包含您已指定為委派系統管理員的 Microsoft 合作夥伴。

系統管理員可以：

-   管理服務管理及帳戶維護

-   提交服務要求給 Microsoft

-   提供使用者帳戶設定和配置供使用者存取服務

-   管理用戶端連線及軟體安裝問題

-   管理您組織內服務可用性的問題

-   使用 Microsoft 的自助支援資源以解決支援問題

如果進行疑難排解之後，問題仍然存在，Microsoft 支援可提供協助。

### Microsoft 支援如何提供協助
我們的角色是：

-   疑難排解、管理和解決客戶問題和擴大議題

-   收集並驗證與特定服務要求相關的資訊

-   提供問題的協調和解決方案管理

-   維持與您的系統管理員通訊，可確保各項問題都能持續得到解決。

-   提供授權、開發票和訂閱查詢等相關事宜的協助

-   提供採購和試用版查詢等相關事宜的協助

-   持續收集客戶對於如何透過調查來提升服務品質的意見反應
