#  Windows Phone 8.0 公司入口網站應用程式已被取代

截至 2016 年 9 月為止，Windows 8.0 公司入口網站應用程式已被取代。 已註冊的 Window 8.0 裝置會維持管理，但無法註冊任何新的裝置。
