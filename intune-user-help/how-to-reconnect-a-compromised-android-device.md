# <a name="how-to-reconnect-a-compromised-android-device"></a>如何重新連接遭盜用的 Android 裝置

IT 管理員偵測到您正在使用根裝置。 「Root 破解」表示裝置已變更成啟用作業系統存取，這可能危及安全性並對公司資源造成威脅。 發生這種情況時，您的裝置就無法繼續存取公司資源。

是否仍需要協助？ 請連絡您的 IT 系統管理員 (請查看[公司入口網站](http://portal.manage.microsoft.com)以取得連絡資訊)，或是傳送電子郵件給 <a href="mailto:wintunedroidfbk@microsoft.com?subject=I'm having trouble with a rooted device&body=Describe the issue you're experiencing here.">Microsoft Android 小組</a>。
